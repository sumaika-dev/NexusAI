# NexusAI — Autonomous Data Intelligence & Predictive Analytics Platform 🚀

NexusAI is a modern, highly interactive, and responsive predictive analytics landing page. This application is designed to showcase advanced data architecture speed, dynamic workflow tracking, and high-end modern UI aesthetics.

---

## ✨ Features

- **Dynamic Custom Cursor**: A unique desktop cursor engine with interactive hover states and fluid animations[cite: 2, 3].
- **GSAP Orchestrated Animations**: Smooth initial hero loads, magnetic buttons, and highly controlled scroll-triggered animations[cite: 2].
- **Tailwind CSS & CSS Grid Bento Layout**: A modern Bento grid system built to display metrics and edge latency statistics cleanly[cite: 1, 3].
- **Card-Glow Effect**: A JavaScript-driven, real-time mouse-tracking radial glow overlay on Bento cards[cite: 2, 3].
- **Fully Responsive Navigation**: A seamless user experience optimized across all screen dimensions.

---

## 🛠️ Tech Stack

- **HTML5** & **Tailwind CSS CDN** (For layout and utility styling)[cite: 1]
- **Custom CSS3** (For custom gradients, glassmorphism, animations, and custom cursor configurations)[cite: 3]
- **JavaScript (ES6)** (For interaction handling, magnetic physics, and coordinate tracking)[cite: 2]
- **GSAP (GreenSock Animation Platform)** & **ScrollTrigger Plugin** (For smooth scrolling triggers and synchronized timelines)[cite: 2]
- **Font Awesome Icons** & **Google Fonts (Plus Jakarta Sans)**[cite: 1]
